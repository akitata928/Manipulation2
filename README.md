# 🛡️ 現代認知戰藍圖：政治操弄與社會工程的五層架構
*(The Blueprint of Modern Cognitive Warfare)*

## 📖 專案簡介
本專案探討現代政治中的「認知戰（Cognitive Warfare）」，並將其背後的社會工程與心理操弄機制轉化為具備維基百科風格的互動網頁。

在現代政治角力中，認知戰的最高境界已不再是單純的「論述戰（說服選民）」，而是演變成「泥巴戰與認同戰」。其終極目標是摧毀大眾對客觀事實的信任，利用演算法與情緒劫持，將人類的大腦當作終端設備進行「駭入與改寫」，最終迫使選民依照設定好的腳本互相攻擊。

本專案結合了**社會心理學**與**數學邏輯模型**（如資訊熵、馬可夫鏈），系統性地解構這套「洗腦演算法」，幫助大眾建立「後設認知（Meta-cognition）」，奪回獨立思考的自主權。

## ✨ 核心特色與功能
* **維基百科風格排版**：乾淨、中立且高易讀性的版面設計，專注於知識傳遞。
* **原生平滑滾動 (Smooth Scrolling)**：點擊目錄連結時，頁面平滑過渡至指定章節。
* **動態目錄監聽 (ScrollSpy)**：向下捲動閱讀時，側邊目錄會自動高亮標示當前所在章節。
* **在地化政治語境**：內容深度結合台灣當前網路生態與時事現象（如：網軍側翼出征、販賣芒果乾、帶風向、內容農場洗版等），讓理論更貼近現實體驗。
* **數理模型對照**：將抽象的心理操弄對應至集合論、預期效用、資訊理論與圖論，提供更理性的分析視角。

## 🏗️ 認知戰五層架構總覽
1. **藍圖設計（掌握資訊制空權）**：透過議程設定限縮大眾的討論範圍。
2. **誘捕機制（情緒動員與劫持）**：喚醒原始生存焦慮與群體歸屬感。
3. **認知故障（真相解構與資訊熵極大化）**：用謊言噴泉與烏賊戰術癱瘓理性運算。
4. **側翼執行（部落主義武器化）**：將被洗腦的受眾轉化為免費的攻擊節點。
5. **心理囚禁（工程化服從與寒蟬效應）**：利用沉默螺旋完成最終的社會控制
